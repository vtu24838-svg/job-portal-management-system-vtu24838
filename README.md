# 💼 CareerConnect

CareerConnect is a full-stack web application designed to connect **job seekers** and **employers** on a single platform. It enables efficient job posting, application tracking, and recruitment management.

## 🚀 Features

### 👨‍💼 Job Seeker

* Register and login securely
* Browse available job listings
* Apply for jobs
* Track application status

### 🏢 Employer

* Register and login
* Post new job openings
* View applicants
* Manage job postings

### 🔐 Admin

* Manage users (job seekers & employers)
* Monitor platform activity
* Maintain system data

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/CareerConnect.git
```

### 2. Open project in IDE

* IntelliJ / VS Code / Eclipse

### 3. Configure Database

Update `src/main/resources/application.properties`:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/careerconnect_db
spring.datasource.username=root
spring.datasource.password=YOUR_PASSWORD
```

### 4. Run the application

* Run `CareerConnectApplication.java`
